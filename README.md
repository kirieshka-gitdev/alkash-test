# 🍺 АлкоТест

Интерактивный тест на трезвость с нейросетевой диагностикой, капчей и токен-доступом.

## ✨ Возможности

- 📊 **20+ наводящих вопросов** с адаптивной системой
- 🎮 **Модульная игра на память** для проверки реакции
- 🔐 **Токен-доступ** для восстановления аккаунта в любом браузере
- 📈 **История и график динамики** ваших состояний
- 👥 **Мультиаккаунты** (до 5 собутыльников)
- 🧠 **Умная капча** с буфером рекламы
